# Conventional Subjects

**A specification for clear, predictable email subject lines**

Stop guessing what emails need your attention. Conventional Subjects brings structure to your inbox with simple, scannable tags.

## Quick Example

**Before:**
```
Re: Important - need your input
Follow up on that thing
Can you review this?
Vendor contract
```

**After:**
```
[ACTION] Approve Q2 vendor contract | Due Mar 15
[ACTION:REVIEW] Feedback on onboarding checklist | Due Mar 22
[UPDATE] Vendor selection finalized
[FYI] Office closed Monday
```
