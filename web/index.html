üõë When NOT to Tag

Do not tag:

  • Casual 1:1 messages
  • Active thread replies
  • Calendar invites
  • Very small audiences
  • Sensitive/emotional topics
  • Micro-updates
Rule of thumb
If removing the tag does not create ambiguity, don‚Äôt use one.
